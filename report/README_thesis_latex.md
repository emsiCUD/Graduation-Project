# Thesis LaTeX scaffold

Files:

- `thesis.tex` — main document, 8 chapters + abstract + appendix.
- `references.bib` — BibLaTeX entries for ViHSD, PhoBERT, PhoW2V, plus the
  standard ML/NLP references.
- Figures: pulled from `../results/figures/` via `\graphicspath{}`.

## Compile

The scaffold targets **XeLaTeX** (Unicode-native, handles Vietnamese
diacritics cleanly).

```bash
cd report/
xelatex thesis.tex
biber thesis
xelatex thesis.tex
xelatex thesis.tex
```

On Overleaf: set the compiler to **XeLaTeX** in Menu → Settings → Compiler,
and BibLaTeX (Biber) is the default.

### Falling back to pdflatex

If your university template mandates pdflatex, swap the preamble's
`fontspec` / `polyglossia` block for:

```latex
\usepackage[utf8]{inputenc}
\usepackage[T5,T1]{fontenc}
\usepackage[main=english,vietnamese]{babel}
\usepackage{lmodern}
```

…and remove the `\setmainfont` / `\setmonofont` lines.
